Takes a Url with a scheme of “http”, “https”, “file”, or “lib” and determine where the content should be loaded from.

Url could refer to: - a specific file or flow (that may or may not exist) - a directory - if exists then look for a provider specific default file - a file in a library, transform the reference into a Url where the content can be found
