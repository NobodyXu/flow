Crate flowr[−][src]

#![deny(missing_docs)] flowr is the flow runner. It reads a flow Manifest produced by a flow compiler, such as `flowc`` that describes the network of collaborating functions that are executed to execute the flow.

Use flowr or flowr --help or flowr -h at the comment line to see the command line options
