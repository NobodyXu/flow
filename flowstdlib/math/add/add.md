## Add (//flowstdlib/math/add)
Add two inputs to produce a new output

#### Include using
```
[[process]]
alias = "add"
source = "lib://flowstdlib/math/add"
```

#### Inputs
* `i1` - first input of type `Number`
* `i2` - second input of type `Number`

#### Outputs
* Sum of `i1` and `i2` of type `Number`
