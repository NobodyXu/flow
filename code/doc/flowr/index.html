flowr is the flow runner for the terminal. It reads a flow Manifest produced by a flow compiler, such as `flowc`` that describes the network of collaborating functions that are executed to execute the flow.

Use flowr or flowr --help or flowr -h at the comment line to see the command line options

The context module implements a set of ‘Context’ functions used by a runtime for flow execution. This particular implementation of this set of functions is a “CLI” one that interacts with the terminal for STDIO and the File system for files.
