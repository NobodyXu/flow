Crate flowc

source ·
Expand description

flowc is a “flow compiler” that takes a hierarchical description of a flow using [flow definitions][flowcore::model::flow_definition::FlowDefinition], [function definitions][flowcore::model::function_definition::FunctionDefinition] and [process references][flowcore::model::process_reference::ProcessReference] and compiles it down into a graph of [runtime functions][flowcore::model::runtime_function::RuntimeFunction] in a [flow manifest][flowcore::model::flow_manifest::FlowManifest] for execution by a flow runner.

Run flowc --help or flowc -h at the command line for a description of the command line options.
